Project overview:

timednotes

timednotes on github

Overview

I built timednotes with the intention of using it to keep track of various job applications. It's a .Net WPF app which lets you create notebooks that store time-stamped notes in a file. A great use case would be keeping simple notes for different classes in school.

Difficulties

Two way data binding in a WPF application was one of the first things that gave me issues, it took me a while to figure out that I had to use an ObservableCollection<T> for the binding to work properly. But once I figured it out it was simple to add code that opened, changed, saved, and closed the notebook file when you changed data in the view.

Getting the program to create a binding from a list of files was another challenge. I ended up creating an observable file list class that implements the INotifyPropertyChanged interface. Internally it uses a FileSystemWatcher to listen for changes to files in a specified directory. Afterwards I could use a directory of files as an ObservableCollection.

Future

In the future I'm going to update the serialization to include more elements like checkboxes, truthy values and other stuff.
