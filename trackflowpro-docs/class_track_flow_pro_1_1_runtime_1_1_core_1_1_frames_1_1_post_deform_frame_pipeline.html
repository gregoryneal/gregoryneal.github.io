Single smoothing step:

  • capture raw deformed frames (qRaw)
  • build smooth transported frames (qSmooth) from position tangents (+ optional hints)
  • blend: qOut = slerp(qRaw, qSmooth, frameSmooth01)
  • overwrite poly.R with qOut
