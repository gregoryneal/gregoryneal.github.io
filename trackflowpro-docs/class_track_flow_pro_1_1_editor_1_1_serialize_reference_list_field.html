Detailed Description

Generic [SerializeReference] list UI for polymorphic managed objects.

  • Adds new items by selecting a derived type from a GenericMenu and setting managedReferenceValue
  • Lets per-type PropertyDrawers render the body
  • Keeps itemsSource in sync with the array size
